⚽ Otimização de Treino

Como o sistema calcula a build perfeita?

1. Como funciona o OVR
Cada atributo tem um peso diferente no OVR.

Jogador de Linha:
Principal vale 60% do OVR
Secundário vale 30% do OVR
Terciário vale 10% do OVR

Exemplo:
Se você aumentar:
+1 Principal → +0.6 no OVR
+1 Secundário → +0.3 no OVR
+1 Terciário → +0.1 no OVR

Ou seja:
O Principal sobe o OVR muito rápido.
O Terciário sobe o OVR bem devagar.

2. Qual o problema de treinar errado?
Se você treinar só o Principal, o OVR sobe muito rápido e você atinge o limite sem conseguir aumentar muito os outros atributos.

Então a estratégia correta é treinar mais o Secundário e principalmente o Terciário, porque eles aumentam o OVR mais devagar e permitem que o jogador fique com mais atributos no total antes de atingir o OVR máximo.

3. O que o sistema faz?
O sistema calcula exatamente quantos pontos faltam para chegar no OVR desejado e testa várias combinações de treino até encontrar a combinação onde:
- O OVR fica exatamente no valor desejado
- A soma dos atributos é a maior possível

Ou seja, ele encontra a build que deixa o jogador o mais forte possível sem passar do OVR.

4. Para Goleiro
Goleiro só usa dois atributos:
Principal vale 90% do OVR
Secundário vale 10% do OVR

Exemplo:
+1 Principal → +0.9 no OVR
+1 Secundário → +0.1 no OVR

Então para goleiro é ainda mais importante treinar o Secundário, porque o Principal sobe o OVR muito rápido.

Resumo:
- Principal aumenta muito o OVR
- Secundário aumenta médio
- Terciário aumenta pouco
- O sistema distribui os treinos para maximizar os atributos sem estourar o OVR
